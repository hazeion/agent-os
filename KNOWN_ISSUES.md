# Known Beta Issues

This is the public, redacted list of active Mentat beta problems and safe
workarounds. The maintainer owns it and reviews it at least twice a week, before
each invitation wave, and before publishing a new release candidate.

There are no active known beta issues yet. The limited cohort has not started.

## What belongs here

- every confirmed P0 or P1 issue that is safe to describe publicly;
- repeated or high-impact P2/P3 issues that testers should know about;
- the affected version, status, safe workaround, public issue link, and fixed
  version when known.

Possible vulnerabilities and reports containing private data stay in
[private security advisories](https://github.com/hazeion/agent-os/security/advisories/new).
This list may say that a private issue is being handled, but never copies its
details. Credentials, endpoints, hostnames, IP addresses, private content, and
participant identities do not belong here.

Resolved entries move to a dated **Resolved** section after the fixed release
is available. They are not silently deleted. If an issue report is disproven,
the entry records that outcome and links the public explanation when safe.
