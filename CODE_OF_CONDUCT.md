# Code of Conduct

Mentat uses a simple rule: be kind, constructive, and respectful.

Welcome people of different backgrounds and experience levels. Discuss ideas
and evidence, not personal traits. Harassment, threats, discrimination,
doxxing, and publishing someone else's private information are not acceptable.

Report conduct concerns privately through the repository's
[security advisory form](https://github.com/hazeion/agent-os/security/advisories/new).
The maintainer may edit or remove contributions, comments, or participation that
break these expectations. Because this is a single-maintainer project, there is
no separate appeals body; GitHub's own policies and reporting tools still apply.
